# 🔐 SOC Password Audit Framework

A **SOC-grade password auditing tool** designed to simulate enterprise security controls and adversarial techniques.

This project demonstrates:
- Blue-team defensive thinking
- MITRE ATT&CK mapping
- Brute-force resistance analysis
- Breach exposure detection using HaveIBeenPwned (k-Anonymity)

---

## 🚀 Features

- 🔍 **HaveIBeenPwned API integration** (safe k-Anonymity)
- 🏢 **Active Directory / LDAP password policy simulation**
- 🔐 **Brute-force resistance scoring**
- 🧠 **Password entropy calculation**
- 🧭 **MITRE ATT&CK technique mapping**
- 📊 SOC-style risk classification (LOW / MEDIUM / HIGH)
